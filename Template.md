## Transaction Limits & Security

- **Daily Transaction Limit:** Users can only send a maximum of ₦100,000 per day.
- **Withdrawal Limit:** Users can only withdraw a maximum of ₦50,000 per day.
- **Security:** All transactions must be verified with a PIN, SMS OTP, or fingerprint.

## Creation of Financial Allocation

Users will have the ability to create a financial Allocation based on the below features:

1.  **Name:** The user will have the ability to name their financial Allocation. 
2.  **Amount:** The user will have the ability to set the amount for financial Allocation. 
3.  **Duration:** Daily Spending allocation. 


### Example of a Daily Spending Allocation

"imagine for TRansport for mon-fri"

1.  **Budget Setup** - User tells the app: *"I need ₦1,000 every weekday for transport, so the total money needed for this setup is ₦20,000."*
2.  **Smart Release** - The app locks the user's money but automatically sends ₦1,000 to their spending wallet every Monday morning (and ₦1,000 every Tuesday morning, etc.).
3.  **The Catch** - If the user tries to withdraw a big chunk of money on a Saturday, the app blocks it, reminding them: *"Remember, your transport money for the weekend is already spent!"*
